Welcome to the Atari Bot website

You can use this Link to add Atari Bot to your discord server When Atari joins the server it will create the role “Atari”. Please to not assign this role to any members you do not trust, as this role will give members full ADMINISTRATOR permmissions on your server.

Default prefix

'ai!'

List of commands

- ai!help - displays the command list
- ai!llama - displays an image of a llama
- ai!turkey - displays an image of a turkey
- ai!oldman - displays an image of a old man
- ai!avatar - displays your discord avatar/profile picture
